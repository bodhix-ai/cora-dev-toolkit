# CORA Development Toolkit - Memory Bank

This directory contains AI context files for developing and maintaining the CORA Development Toolkit.

## Purpose

The memory-bank provides persistent context for AI agents working on:

- Toolkit documentation
- Project/module templates
- Validation framework
- Migration from pm-app repos

## Files

| File               | Purpose                           |
| ------------------ | --------------------------------- |
| `activeContext.md` | Current focus and session context |
| `progress.md`      | Implementation progress tracking  |
| `techContext.md`   | Technical decisions and patterns  |
